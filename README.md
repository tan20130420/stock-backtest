# 股票三分之一投资法回测模拟器

基于「三分之一投资法」的 A 股回测工具。

## 功能特性

- 📈 **K 线图 + 买卖标记**：直观展示买卖点位
- 🔥 **参数热力图**：枚举所有 (x, y) 组合，找到最优买跌/卖涨幅度
- 📊 **收益曲线**：对比策略收益 vs 持有不动
- 📋 **交易明细**：完整买卖记录
- 🏦 支持 A 股、ETF 基金、可转债

## 三分之一投资法规则

1. 将本金分成 3 份（每份约 1/3）
2. 当价格从基准价**下跌 x%** 时，买入 1 份
3. 当价格从平均成本**上涨 y%** 时，卖出 1 份
4. 系统自动枚举所有 x∈[1%,20%], y∈[1%,30%] 组合，找出达到目标收益率的最优参数

## 快速开始

```bash
pip install flask flask-cors
python server.py
```

浏览器访问 `http://localhost:8080`

## 技术栈

- **前端**：HTML5 + ECharts 5 + 原生 JS（无需构建）
- **后端**：Python Flask（仅作静态文件服务）
- **数据源**：浏览器直连东方财富 API（实时行情）

## 使用说明

1. 输入证券代码（如 `600519`、`510300`、`110043`）
2. 选择回测日期范围
3. 设置初始资金（默认 10 万）
4. 设置目标收益率（如 10%）
5. 点击「开始回测」

> 数据来源：东方财富，仅供学习研究，不构成投资建议。
